# Poc tosspayments  


## install

```
yarn add @tosspayments/payment-sdk  

# 참고) 결제 위젯 > https://docs.tosspayments.com/guides/payment-widget/overview  
- 노코드 툴로 관리가 가능하다. 하지만 테스트를 위해 사업자 등록증이 필요
yarn add @tosspayments/payment-widget-sdk
```
